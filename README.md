# Trading Journal (SwingTrader?)

Trade journal app for tracking swing trades. Built with React/FastAPI.

## Features

- Trade tracking
- Basic analytics
- Chart uploads
- CSV import

## Setup

```bash
# Backend
cd backend
pip install -r requirements.txt
uvicorn app.main:app --reload (should also prob specify port)

# Frontend  
cd frontend
npm install
npm run dev
```
